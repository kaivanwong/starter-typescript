# Starter TypeScript

Starter template for TypeScript library.

## Usage

Clone the git repository.

```sh
git clone https://github.com/kaivanwong/starter-typescript.git

cd starter-typescript
```

Use `pnpm` for package management.

```sh
npm i pnpm -g

pnpm i
```
